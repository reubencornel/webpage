Glossary

STL Standard Template Library is a library of most commonly used template classes and algorithms. I suggest you read up on this as this could come in useful anytime. I read this from the The C++ Complete Reference by Herbert Schildt.



segmentation fault This occurs because a program violates its memory restriction, by trying to read or write to a part of the memory that does not belong to it. This page tell you how to debug seg faults



AWK It is an acronym for Aho, Weinberger and Kernighan. This is a small text processing language with C syntax. It mainly used to process files which have a structured format.



sed Stands for stream editor. It is a extremely powerful tool for unix text processing although it commands look like it was created in the age of the dinosaurs.



perl Is acronym for Practical Extraction and Report Language. I would like to call it the swiss army knife langauge. If you are using GNU/linux and want an intro on the language type 'perldoc perlintro' at the shell prompt.



GRUB The Grand Unified Boot Loader. This is an amazingly powerful bootloader. Very easy to configure, configuration file /etc/grub.conf



core file This is a dump of the contents of the memory locations that hold your program. In the days of yore, the RAM was referred to as core since most of it was magnetic core memory.
